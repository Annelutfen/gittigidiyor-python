Base Interface for GittiGidiyor RESTLIKE API. Every service provider must implement this interface. Services can be anonymous or community services. Services that implements this interface are;

* Application service

* Category service

* City service

* Developer service

* Catalog service

* Search service

* Product service

* Sale service

* Cargo service

* User messages service

* Authentication service
