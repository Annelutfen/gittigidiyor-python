source code 

Performs the 'sendCargoInformation' API method for the Cargo Service API of Gittigidiyor.

sessionId: session key fo the logged in user.

saleCode: sale code.

cargoPostCode: Cargo number.

cargoCompany: cargo company such as yurtici, MNG..

cargoBranch: township for the cargo branch such as Taksim, Kadikoy.

followUpUrl: cargo external url.

userType: Determines the user. Can be 'A' for all, 'S' for seller, 'B' for buyer or 'N' for none.

inputCT: input content type (xml or json)

outputCT: output content type (xml or json)

lang: 'tr' or 'en'
